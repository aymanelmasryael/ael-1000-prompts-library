**A comprehensive AI prompts library** with a built-in generation engine, 10 curated batches of 100 prompts each, smart search, and multi-format export.

## Features

- **1,000+ Prompts**: 10 batches × 100 prompts each across 20+ professional roles
- **Generation Engine**: AEL Prompt Generation Engine for dynamic prompt creation
- **Smart Search**: Real-time filtering across all prompts by keyword or role
- **10 Batches**: Curated, sequential batches for organized browsing
- **3 Export Formats**: JSON, CSV, TXT — export the entire library
- **Ultra-Lock System**: Premium prompts with gold lock indicators
- **One-Click Copy**: Copy any prompt to clipboard
- **Role-Based Organization**: Product Managers, Data Analysts, UI Designers, DevOps, ML Engineers, and more
- **Glassmorphism UI**: Dark theme with blue (#0074FF) accents

## Tech Stack

- **HTML5** — Semantic structure
- **CSS3** — Glassmorphism, flexbox, grid, custom properties
- **JavaScript** — Generation engine, search engine, 3 export formatters, Ultra-Lock system
- **Deployment**: GitHub Pages

## Usage

1. Browse through 10 curated batches
2. Search across all prompts by keyword
3. Click a prompt card to copy it to clipboard
4. Use the Generation Engine to create new prompts
5. Export the entire library as JSON, CSV, or TXT
6. Premium prompts show with gold Ultra-Lock (🔒)
